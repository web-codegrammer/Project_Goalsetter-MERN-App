# Goalsetter MERN app

This is the Goalsetter MERN Stack App with the implementation of REST API(CRUD Operations) in it.

## Usage

Rename the .envexample to .env and add your MONGO_URI

### Install dependencies

```
# Backend deps
npm install

# Frontend deps
cd frontend
npm install
```

### Run Server

```
npm run server
```
